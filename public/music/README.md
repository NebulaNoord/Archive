# Drop your music here

The Media Player plays real audio files from this folder. Name them exactly:

- `swimming-pools.mp3`  → Kendrick Lamar — Swimming Pools (Drank)
- `champion.mp3`        → Kanye West — Champion
- `hood-gone-love-it.mp3` → Jay Rock ft. Kendrick Lamar — Hood Gone Love It

If a file is missing, the player falls back to a short synth preview so the
jukebox still makes sound. Any MP3 here is loaded automatically — add more and
they'll appear next time you open the Media Player (edit `src/data/lore.ts`
`musicTracks` to add titles/artists).

Note: don't commit copyrighted audio to the repo. These are local-only.
