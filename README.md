# PixelOS

PixelOS is a modular operating-system-style portfolio for Kayden. It uses React, TypeScript, Vite, and Tailwind CSS.

## Foundation included

- Animated boot sequence
- Desktop shell with wallpaper, icons, taskbar, clock, notification card, and window manager
- Window open/close/minimize/maximize/drag behavior
- Lazy-loaded apps through an app registry
- About, Projects, Arcade, Terminal, Settings, File Explorer, and Browser app modules
- Developer mode with FPS, open window count, theme, and React info

## Architecture rule

PixelOS is an OS shell and launcher. Large projects and games must remain separate applications and should be linked or loaded externally.

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
```
